Nerladdningar

Getting started

Är du ny? Ta en titt på vår Getting Started, FAQ, och Tutorials-sidor på wiki:n (på engelska).
Annars kan du se ändringshistoriken.

Du kan möjligtvis också vilja se på några spel. Spel förser den grundläggande spelmiljön för motorn att köra med Lua-skript. Olika spel har olika objektiv, såsom överlevnad, byggning eller Player vs Player. Du kan sedan lägga till moddar utöver ett spel för att anpassa din upplevelse ytterligare.

Windows

Fungerar på Windows 8, 8.1 och 10. Kan fungera på äldre versioner men stöds inte officiellt.

  • Luanti 5.10.0 - portabel, 64-bit (recommended)
  • Luanti 5.10.0 - portabel, 32-bit

Luanti distribueras som portabel .zip-fil på Windows. Extrahera den till en mapp där du har skrivbehörighet och starta minetest.exe som finns inom mappen bin.

Du kan även få de senaste utvecklingsversionerna av Luanti från byggen gjorda av gemenskapsmedlemmar. Dessa versioner är nyare än den officiellt lanserade versionen och innehåller nya funktioner (men kan däremot vara ostabil).

Android

Android 6 eller senare rekommenderas.

Alternativt, ladda ner APK-filen:

  • ARM: 64-bit - 32-bit
  • x86: 64-bit - 32-bit

Notera: Vi rekommenderar inte att använda inofficiella versioner som oftast finns på Playbutiken. De kan innehålla excessiva mängder reklam eller spionprogramvara, eller distribueras som proprietär programvara.

Linux

Paket

Dessa kan vara mer eller mindre föråldrade. Om de är för föråldrade borde du bygga från källkoden.

  • Flatpak (all distributions) - Stable
  • Ubuntu - Stable PPA or Daily PPA, as the repos are outdated
  • Debian - use Flatpak or compile from source, as the repos are outdated
  • Arch Linux - Stable, Unstable (AUR)
  • Fedora - Stable
  • openSUSE - Stable
  • Mageia - Stable
  • Gentoo - Stable
  • Void Linux - Stable
  • Alpine Linux - Stable
  • NixOS - Stable

FreeBSD

Package manager

On FreeBSD 9.1 and newer, you can use the official pre-built packages: pkg install minetest

Compile using ports

You can also compile Minetest and choose build options easily using the FreeBSD port dialogs.

cd /usr/ports/games/minetest
make install

macOS

macOS 11.3 eller senare är rekommenderat. Kan fungera på äldre versioner om man bygger från källkoden.

  • Signed Apple Silicon - Luanti 5.10.0 - App
  • Signed Intel 64-bit - Luanti 5.10.0 - App
  • Use Homebrew to get the stable version:
    brew install minetest
  • MacPorts also contains a port of the stable version:
    sudo port install minetest
